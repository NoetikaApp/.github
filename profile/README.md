# 🧠 Noetika Organization

> *The future of wisdom.*

Welcome to the official **Noetika** GitHub organization.  
Noetika is an ecosystem of AI-powered applications focused on philosophy, critical thinking, self-reflection, and human-centered technology.

Our mission is to combine modern artificial intelligence with the timeless wisdom of philosophy to create meaningful digital experiences across Web, iOS, and Android.

---

# 🌍 Vision

Technology should not only optimize productivity — it should also help people think, reflect, and grow.

Noetika explores a new category of software:
- Philosophical AI systems
- Reflective conversational interfaces
- Wisdom-driven user experiences
- Human-centered artificial intelligence

---

# 🚀 Main Projects

| Project | Description |
| :--- | :--- |
| **Noetika Web** | Main Next.js platform for philosophical AI interaction |
| **Noetika API** | FastAPI backend powering AI, RAG, moderation, and accounts |
| **Noetika Android** | Native Android application built with Jetpack Compose |
| **Noetika iOS** | Native Apple ecosystem app built with SwiftUI |
